Formularios


Un formulario HTML es una sección de un documento que contiene contenido normal, código, elementos especiales llamados controles (casillas de verificación (checkboxes), radiobotones (radio buttons), menúes, etc.), y rótulos (labels) en esos controles. Los usuarios normalmente "completan" un formulario modificando sus controles (introduciendo texto, seleccionando objetos de un menú, etc.), antes de enviar el formulario a un agente para que lo procese (p.ej., a un servidor web, a un servidor de correo, etc.)


El elemento "form" El elemento "form" define formalmente un formulario y los atributos que determinan el comportamiento del formulario. Cada vez que desees crear un formulario HTML, debes empezar utilizando este elemento y anidando todo el contenido dentro de él. Muchas tecnologías de asistencia y complementos del navegador pueden descubrir elementos "form" e implementar códigos de apoyo (hooks) especiales para que sean más fáciles de usar. Los nuevos atributos para las etiquetas "input" más habituales e importantes son:

  1. required: El campo es ogligatorio.
  2. autofocus: Inicialmente el campo está seleccionado (sólo puede haber uno por formulario).
  3. placeholder: Se muestra un valor por defecto en gris que se elimina cuando el usuario introduce algún valor.
  4. autocomplete: Permite el rellenado automático del "input" en base al texto que vaya escribiendo el usuario, según los valores introducidos en ocasiones anteriores.
